Raccourcis clavier

Dans Vienna vous pouvez utiliser votre clavier pour exécuter rapidement beaucoup d'actions. Pour trouver les raccourcis clavier pour les commandes courantes, regardez dans les menus (ou parcourrez la liste en bas de cette page). Beaucoup d'objets dans Vienna, tels que la liste des dossiers ou celle des articles possèdent aussi un menu contextuel. Pour afficher un menu contextuel, pressez la touche "Ctrl" et cliquez sur l'objet.

Pour réaliser une action, pressez les raccourcis indiqués ci-dessous.

Combinaison de touches Action
Raccourci à une touche  
f Sélectionne le champ de recherche.
r Permute l'état lu/non lu pour les articles sélectionnés.
s Marque tous les articles du dossier courant comme étant lus et passe au prochain dossier contenant des articles non lus.
k Marque tous les articles du dossier courant comme étant lus.
m Permute l'état affiché/masqué du drapeau pour les articles sélectionnés.
< Affiche l'article ou la page web précédent.
> Affiche l'article ou la page web suivant.
Barre d'espace Passe au prochain article non lu, à moins qu'il reste du texte à afficher pour l'article en cours ; dans ce cas, on affiche la page de texte suivante.
Commande-U Passe au prochain article non lu.
Entrée/Return Ouvre la page web source de l'article sélectionné.
Delete/Backspace Déplace les articles sélectionnés dans la corbeille. Lorsque cette commande est utilisée à partir de la corbeille, les articles sélectionnés sont définitivement détruits.
Raccourcis généraux  
Commande-N Crée un nouvel abonnement.
Shift-Commande-G Create a group folder.
Shift-Commande-F Create a smart folder.
Shift-Commande-E Edit the selected folder URL or edit a smart folder criteria.
Shift-Commande-D Delete the selected folder.
Shift-Commande-N Rename the selected folder.
Commande-R Rafraîchit tous les abonnements.
Shift-Commande-R Rafraîchit uniquement les abonnements sélectionnés dans la liste des dossiers.
Commande-Minus Stoppe tout rafraîchissement en cours.
Commande-P Imprime les articles sélectionnés.
Shift-Commande-P Affiche la fenêtre de mise en page.
Shift-Commande-V Validate the selected feed.
Command-? Affiche l'aide Vienna.
Raccourcis Articles  
Shift-Commande-M Marque d'un drapeau les articles sélectionnés.
Shift-Commande-U Marque comme étant lus les articles sélectionnés.
Shift-Commande-S Marks the current folder read then skips to the next folder with unread articles.
Shift-Commande-O Restores an article from the trash back to its original folder.
Shift-Commande-K Marque tous les articles des dossiers sélectionnés comme étant lus.
Alt-Commande-K Marks all articles in all folders read.
Raccourcis fenêtre  
Commande-W Ferme l'onglet actif si plusieurs sont ouverts ; sinon, ferme la fenêtre Vienna.
Alt-Commande-W Ferme tous les onglets autres que la liste des articles. Note : cette commande remplace la commande "Fermer l'onglet" dans le menu Fichier lorsque la touche Alt est pressée.
Control-Commande-Gauche Affiche l'onglet précédent.
Control-Commande-Droite Affiche l'onglet suivant.
Shift-Commande-W Ferme la fenêtre Vienna, mais conserve l'application active en tâche de fond. vous pouvez réafficher la fenêtre en cliquant sur l'icône de Vienna dans le dock ou en pressant Commande-1.
Commande-M Réduit la fenêtre principal ves le dock.
Commande-0 Affiche la fenêtre l'activité.
Commande-1 Réaffiche la fenêtre Vienna si celle-ci a été préalablement fermée.
Commande-2 Affiche la fenêtre de téléchargement.
Racourcis du navigateur internet  
Alt-R Recharge la page web active.
Command-Virgule Arrête le chargement de la page web active.
Command-[/Command-Gauche Affiche la page web précédente.
Command-]/Command-Droite Affiche la page web suivante.
Command-F Acitve le champ de recherche. Après validation par Enter, le texte saisi sera recherché sur la page web active.
Command-G Recherche l'occurrence suivante sur la page web active.
